Of course, I should to give credit where it is due. This site uses the Vollkorn, PT Sans, and Inconsolata typefaces hosted by the free Google WebFonts service.

I wrote the most of the css for this site myself, but used the file "reset.css" from an old version of the css blueprint project. The rest was heavily inspired by Emmanuel Beffara's website and bears an obvious resemblance. The colors are taken from the "solarized" palette by Ethan Schoonover.
